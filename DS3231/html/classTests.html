Detailed Description

///The Tests class to test DS3231 functions

/// This class inherits form DS3231 and there for requires hwlib to operate This class is written as a tests class and contains tests that will check certain functions of the DS3231 class when the program is flashed to arduino Due This class contains pre-defined test parameters to ensure functionality of the DS3231 class It is best to comment out the test constructor + call in the main program when one wants to use a display with this library since this test writes and reads date registers witch may interfere with normal operations
